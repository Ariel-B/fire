# Code Reviews

This directory is reserved for future review documents.

## Where to save new reviews

Create each review in its own dated folder under `docs/reviews/` using this format:

```text
docs/reviews/YYYY-MM-topic-name/
```

## Minimum contents

- Add a `README.md` that summarizes the review.
- Place any supporting review documents in the same folder.
- Link the new review folder from the relevant documentation only if needed.
